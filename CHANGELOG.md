# Changelog

All notable changes to the "vclutter" extension will be documented in this file.

## [0.0.1] - 2026-02-15
### Added
- Initial release of vClutter.
- **Smart Cleaning Engine**: Removes comments while preserving strings and important notes like `TODO` or `FIXME`.
- **Partial Clean**: Support for cleaning only the selected code area.
- **Batch Clean**: "Sweep Folder" feature to clean multiple files in a directory at once.
- **Indent Fixer**: Automatically corrects indentation for bracket-based languages.
- **Status Bar Integration**: Quick access to cleaning via the broom icon.
- **English Localization**: Professional English UI messages and alerts.
